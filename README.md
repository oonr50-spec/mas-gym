## GitHub Pages
1. Upload all files and folders to the repository root.
2. Open Settings > Pages.
3. Choose Deploy from a branch.
4. Select main and /(root).
5. Save.

## Images
The current version uses premium gradient placeholders so it works without photos.
Later, replace gallery placeholders with real approved photos.
Do not publish member photos, trainer photos, reviews, or before/after stories without permission.

## Google Maps
The map query is approximate. Replace the iframe URL with the exact Google Maps share/embed link when available.

## Static-site booking
The booking form does not save data. It creates a WhatsApp message and opens WhatsApp.
